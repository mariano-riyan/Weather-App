# WeatherHub

A professional weather dashboard built to provide real-time updates and forecasts. This project is designed as a learning milestone, focusing on React fundamentals, state management, and API integration.

## Features
- **Real-time Data:** Integration with OpenWeatherMap API.
- **Geolocation:** Automatically detect local weather based on user location.
- **Search:** Find weather for any city globally.
- **Personalization:** Save favorite cities for quick access.
- **Unit Toggle:** Seamless switching between Metric (°C) and Imperial (°F).
- **Responsive Design:** Mobile-first layout using TailwindCSS v4.

## Tech Stack
- **Framework:** React + Vite
- **Styling:** TailwindCSS v4
- **API:** OpenWeatherMap
- **State Management:** React Context API & useState

## Setup Instructions
1. Clone the repo: `git clone <your-repo-url>`
2. Install dependencies: `npm install`
3. Create a `.env` file in the root: `VITE_OPENWEATHER_API_KEY=your_key_here`
4. Start development: `npm run dev`
